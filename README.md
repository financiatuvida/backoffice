
# FINANCIATUVIDA - Demo

Repositorio demo para el dashboard "FINANCIA TU VIDA".

## Contenido
- Frontend React (Vite)
- Demo de cálculo automático de comisiones (8 niveles)
- Mock API (archivos JSON en public/mock-api/)
- Exportación a Excel (.xlsx)

## Uso (sin Node/Git)
1. Subir carpeta `FINANCIATUVIDA` a un repo en GitHub (Add file → Upload files).
2. Conectar el repo a Vercel y desplegar (Framework: Vite, Build: `npm run build`, Output: `dist`).
3. O puedes abrir los archivos localmente en un editor; para ejecutar necesitarás Node.

## Notas
- Archivos mock en `public/mock-api/` para simular endpoints.
- Exporta a Excel con el botón del header.
